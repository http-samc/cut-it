# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| ``v.0.5.0@beta`` *and lower* | ❌ (no support) |
| ``v.1.0.1@beta``  | ✔️ (full support) |
| ``v.1.0@release``  | ✔️ (full support) |
